# 10. Architecture Diagrams

This directory contains comprehensive architecture diagrams for the Multilingual Legal Conversational Bot.

## Diagrams

- `llm_rag_architecture.txt`: Complete LLM + RAG architecture diagram
- `lora_finetuning_flow.txt`: LoRA fine-tuning flow diagram
- `multi_agent_architecture.txt`: Multi-agent architecture diagram
- `rlhf_training_lifecycle.txt`: RLHF training lifecycle diagram

View these diagrams in any text editor or render them using ASCII art viewers.
